# ¿Cómo instalar todas las dependencias?

### Autor: Annese Agustin. 

1. Clonar el repositorio remoto:

bash
git clone (https://github.com/agusann/Python-Seminario/tree/master/PRACTICA%202)
cd Practica2 master

2. Crear virtual environment y activarlo:
(usando el cmd)

bash
python -m venv venv
.venv\Scripts\Activate.bat

3. Descargar dependencias:

bash
pip install -r requirements.txt


4. Ejecución:
Dentro de la terminal ejecutar "jupyter lab". El archivo a ejecutar se encuentra dentro de la carpeta "notebooks" con la extensión ".ipynb".
